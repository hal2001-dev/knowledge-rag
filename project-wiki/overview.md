## 최근 주요 결정

> [decisions.md](wiki/architecture/decisions.md) 참고

- **ADR-022 (2026-04-23)**: 폴더 단위 일괄 색인 CLI(`scripts/bulk_ingest.py`). 재귀 탐색·L1 중복 스킵·결과 JSON 리포트. 관리자 UI 버튼·API는 인증·공개배포 묶음과 함께 미래 도입
- **ADR-021 (2026-04-22)**: DELETE 고아 파일 정리 + HybridChunker `max_tokens=480` 명시. 토큰 경고 0건, 회귀 0
- **ADR-020 (2026-04-22)**: 인덱스 커버리지 카드 — `/index/overview` 서버 캐시. 빈 채팅에 "이 시스템이 아는 내용" + 예시 질문 5개. 캐시 히트 5ms
- **ADR-019 (2026-04-22)**: 후속 질문 제안 — 단일 LLM 호출에 JSON 구조화로 통합. 답변 후 3개 배지 → 클릭 재질의. 회귀 0, 추가 LLM 호출 0회
- **ADR-016 (2026-04-22)**: Embedding — OpenAI 기본 유지, BGE-M3 토글. TASK-004 프레임워크로 A/B 실시 결과 동률·소폭 하락, 전환 이득 없음. 토글 인프라만 확보
- **ADR-015 (2026-04-22)**: 평가 프레임워크 — Ragas + 자체 Retrieval 벤치. 기반선(Hit@3=1.000, faithfulness=0.886) 수립. 이후 모든 품질 ADR은 before/after 수치 첨부 원칙
- **ADR-014 (2026-04-22)**: LLM 백엔드 토글 인프라 구축. `LLM_BACKEND=openai|glm|custom` 한 줄 변경으로 공급자 전환 가능. 기본 유지는 ADR-013 결론 준수
- **ADR-013 (2026-04-22)**: LLM은 gpt-4o-mini 유지. GLM 계열은 컨텍스트 준수·안전 필터·생태 안정성에서 현재 유리하지 않음. 비용/데이터 주권 이슈가 생기면 `.env` 토글 패턴으로 1시간 내 교체 가능 구조를 유지
- **ADR-012 (2026-04-22)**: Reranker를 `flashrank` → **`bge-m3`** (BGE-reranker-v2-m3)로 채택. 한↔영 크로스 실패 해소, 토글로 회귀 가능

## 예정 태스크 완료 시 사용자 개선점

각 태스크 완료가 **사용자 입장에서 실제로 무엇을 바꾸는지** 정리. "before → after" 형식.

### TASK-007 완료 시 (Phase 1 — 후속 질문 제안)

| 항목 | Before | After |
|---|---|---|
| 답변 후 탐색 | 사용자가 다음 질문을 스스로 떠올려야 함 | 답변 아래에 **"이어서 물어볼 수 있는 질문" 3개 배지**. 클릭 시 즉시 재질의 |

**사용자 한 줄 가치**: "답변만 받고 끝"에서 "답변 + 다음 길 제시"로 진화.

### 조건부 장기 태스크 완료 시 (참고)

| 태스크 | 사용자 개선점 |
|---|---|
| **Graph RAG** (조건 충족 시) | "A 문서의 X가 B 문서와 어떻게 연결되나" 같은 **multi-hop·교차 문서 질의**. 엔티티 네트워크 시각화로 지식 지도 탐색 |
| **하이브리드 검색** | "ROS 2.0.3" 같은 **정확한 용어·숫자·고유명사** 매칭 보장 (현재 벡터 유사도는 이 케이스에 약함) |
| **대화 요약 메모리** | 50턴 넘는 **긴 세션에서도 초기 문맥 유지** (현재는 최근 20턴만 주입) |
| **스트리밍 응답** | 긴 답변 시 **타이핑처럼 실시간 표시**, 지연 체감 ↓ |
| **평가 dataset 확장** | 답변 품질 측정의 신뢰도 ↑ (hold-out 분리로 과적합 방지) |
| **MCP 재개** | Claude Code/Cursor 등에서 **외부 에이전트가 이 RAG를 도구로 호출** 가능 |

## 알려진 한계 (참고)

태스크 현황의 "기술 부채" 블록에서 해소 로드맵을 관리. 아래는 현시점 스냅샷.

- 중복 감지 L1만 — 포맷만 다른 동일 내용은 미감지 (장기 재검토)
- `content_hash=NULL` 레거시 문서 — 백필 스크립트 미작성
- 대화 히스토리 최근 20턴 제한 — 긴 세션 초기 문맥 유실 (요약 메모리 도입 시 해결)
