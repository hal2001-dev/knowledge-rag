## 유저 스토리

### F-001: PDF 업로드

```
As a 사용자
I want to PDF 파일을 업로드하면
So that 내용에 대해 질문할 수 있다

Acceptance Criteria:
- PDF 파일을 드래그앤드롭 또는 파일 선택으로 업로드 가능
- 업로드 후 처리 상태(진행 중/완료/실패)를 확인 가능
- 지원 최대 파일 크기: 50MB
- 지원 형식: PDF (v1.4 이상)
```

### F-003 + F-005: 질문 & 답변

```
As a 사용자
I want to 자연어로 질문을 입력하면
So that 업로드된 문서 기반의 답변을 받을 수 있다

Acceptance Criteria:
- 답변은 5초 이내에 시작되어야 함
- 답변에는 출처 문서 및 페이지 번호 표시
- 관련 문서가 없는 경우 "찾을 수 없음" 명시
```
